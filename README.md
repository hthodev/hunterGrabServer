## Installation

```bash
$ yarn install
```

## Create file .env
```
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_CALLBACK_URL=
MONGODB_URI=
PRIVATE_KEY=  //generate rs256
PUBLIC_KEY=   //generate rs256
```

## Running the app
Run prod env
```
npm start
```

Run dev env
```
npm run dev
```
